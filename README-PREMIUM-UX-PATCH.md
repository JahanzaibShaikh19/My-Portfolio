# Premium UI/UX Upgrade

This branch upgrades the portfolio with a stronger front-end interaction system and a more premium visual language.
  
## Included

- Animated cyber / 3D-like ambient background with aurora glow, particles, perspective grid, depth rings, noise texture, and pointer spotlight.
- Smooth sticky header with scroll progress, active section detection, desktop nav, mobile overlay nav, and theme toggle.
- HCI-focused section that explains visual hierarchy, low cognitive load, micro-interactions, and trust psychology.
- Hero redesign with stronger CTAs, trust chips, proof signals, glass profile card, and interactive service cards.
- Resume redesign with scannable hierarchy and grouped skill panels.
- Portfolio proof filters with smoother Framer Motion transitions and richer hover overlays.
- Contact section redesign with lower-friction UX, trust signals, local draft behavior, and improved CTA feedback.
- Accessibility support through `prefers-reduced-motion`, clear labels, and aria-pressed filter buttons.

## Local check

```bash
pnpm install
pnpm dev
pnpm build
```

## Design intent

The changes are built around HCI and psychological design concepts: reduce cognitive load, guide attention, increase trust through proof signals, make navigation predictable, and use micro-interactions to make the interface feel responsive without becoming heavy.
